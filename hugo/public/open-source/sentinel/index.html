What It Does

Sentinel detects and reports deviations from defined security and compliance baselines, including:

  • Cloud misconfigurations (IAM, S3, logging, encryption)
  • Policy violations in Terraform, Helm, or Kubernetes
  • Missing control evidence in Vaultum
  • IaC PRs lacking approved Signatory signatures
  • Controls declared in Blueprint modules but not enforced
